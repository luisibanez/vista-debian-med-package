MAG*3.0*121 Readme File
=======================

This file contains last-minute information about MAG*3.0*121.

(1) The version of the VistA Imaging System Background Processor User Manual, which is distributed with MAG*3.0*121 supercedes the version that is linked to the Background Processor client application.

(2) Installing the MAG*3.0*121 VistA Imaging Background Processor adds a second Background Processor entry in Windows Add/Remove Programs. This does not affect the functionality of MAG*3.0*121.

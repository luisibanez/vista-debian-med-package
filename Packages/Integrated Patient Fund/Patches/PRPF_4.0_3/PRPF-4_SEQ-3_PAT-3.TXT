Description:
============

 This informational patch, PRPF*4*3, describes the purpose and contents 
 of VPFS maintenance package VPFS-PKG-1.1.3.3.zip.
  
 Since this HealtheVet application is a centralized application and there 
 are no changes to VistA patches, it is important to note that NO software 
 will need to be installed at VistA facilities.
  
 All the fixes are addressed through the VPFS application ear file 
 VPFS-app-1.1.3.003.ear and VPFS database patch 1.0.7.001.More information 
 related to these application and database patches are described in the 
 respective readme and release files.
  
 VPFS patch package VPFS-PKG-1.1.3.3 includes fixes for five Remedy 
 tickets.  Three of these fixes were made in the VPFS application build 
 1.1.3.003, that resides on the centralized server.  Two Remedy ticket 
 fixes were made to a stored procedure and View on the centralized 
 database in the VPFS database patch 1.0.7.001.
  
 The Remedy tickets being fixed in this package are:
 380403 - Filtering Inactive Stations.
 378841- Transaction Integrity Error.
 203253 - Fiscal Audit report Sorting.
 170949 - VPFS Calendar Issue.
 170426 - VPFS Patient records Inactive.

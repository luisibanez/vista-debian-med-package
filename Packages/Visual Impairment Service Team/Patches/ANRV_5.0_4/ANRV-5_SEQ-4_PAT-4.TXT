Subject: RELEASE 5.0.29 INFORMATIONAL PATCH

Category:

Description:
============

                                                                         
 BR patch package 5.0.29 includes fixes for 15 Remedy tickets.
 Out of 15 remedy tickets 2 are database fixes.
 13 remedy tickets fixes were made in the Java application, BR EAR file
 BR-APP-5.0.29.4 that resides on the centralized server.  
 No installations need to be done on the Vista sites
  
 The following is a list of Blind Rehab field test sites:
  
 VA Puget Sound Health Care System
 Long Beach VAMC
 Minneapolis VAMC
 Augusta VAMC          
  
  
 INSTALLATION INSTRUCTIONS
 =========================
  
 This is a Centralized Server promotion from BR 5.0.28 to 5.0.29
 Local sites do not need to install anything.
